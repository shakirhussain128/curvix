# Curvix ⚡️  
**AI-Powered Code Editor built for speed, focus, and developer productivity.**

Curvix is a modern AI-first coding environment designed to help developers write, refactor, debug, and ship faster.  
It combines a clean IDE experience with project-aware AI assistance—similar to Cursor, but built with a focus on simplicity, speed, and real-world workflows.

---

## ✨ Features

- **AI Chat inside the Editor**  
  Ask questions, explain code, generate functions, and get instant help without leaving your workspace.

- **Project-Aware Assistance**  
  Curvix understands your folder structure and context to provide smarter suggestions.

- **Smart Refactoring**  
  Improve code quality with AI-powered refactor recommendations.

- **Code Generation**  
  Generate boilerplate, APIs, UI components, and scripts instantly.

- **Debugging Help**  
  Paste error logs and get guided debugging steps with fixes.

- **Modern Developer UI**  
  Minimal design, keyboard-first navigation, and distraction-free coding.

---

## 🧠 Vision

Curvix aims to become the fastest way to build software by combining:

- a lightweight IDE experience  
- intelligent AI features  
- clean UX for developers  
- powerful integrations (Git, terminal, extensions)

---

## 🛠️ Tech Stack (Planned)

Curvix is built using modern, scalable technologies:

- **Frontend:** React + TypeScript  
- **Editor:** Monaco Editor (VS Code engine)  
- **Backend:** Node.js / Express (or FastAPI for Python services)  
- **Desktop:** Electron.js (planned)  
- **AI:** OpenAI API / compatible LLM providers  
- **Auth:** JWT / OAuth (future)  
- **Database:** PostgreSQL (future)  
- **Hosting:** Vercel / AWS / Railway
